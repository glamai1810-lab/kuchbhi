# 📉 Customer Churn Prediction System

An end-to-end Machine Learning project that predicts whether a customer will leave a service provider based on their demographics, account information, and services used.

## 📁 Project Structure
*   **`backend/`**: Contains the FastAPI server and the Machine Learning model training scripts.
*   **`frontend/`**: Contains the HTML/CSS/JS user interface.
*   **`data/`**: Stores the raw CSV dataset.
*   **`sql/`**: Contains table schemas and analytics queries.
*   **`powerbi/`**: Placeholder for the visual Power BI dashboard.
